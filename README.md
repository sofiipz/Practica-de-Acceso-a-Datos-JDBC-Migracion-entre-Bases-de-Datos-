# Práctica de Acceso a Datos (JDBC)
Proyecto para la asignatura de Acceso a Datos.  
Simula la migración de datos entre dos bases de datos (prac2 y prac2migra) utilizando JDBC.  
Incluye dos versiones del proyecto: una versión inicial (V1) y una versión mejorada (V2).

## Contenido del proyecto
- Creación de tablas en ambos esquemas.
- Carga masiva de datos mediante batch.
- Uso del patrón DAO para operaciones CRUD.
- Migración completa de datos con transacciones.
- Procedimientos almacenados y pool de conexiones en la versión 2.
- Generación de un informe final de migración.

## Tecnologías
- Java
- JDBC
- MySQL
- H2
- HikariCP
