# Practica-de-Acceso-a-Datos-JDBC-Migracion-entre-Bases-de-Datos-
Proyecto de Acceso a Datos (2º DAM) que simula la migración entre dos bases de datos (prac2 y prac2migra) usando JDBC. Incluye creación de tablas, carga masiva con batch, uso de DAOs, transacciones, procedimientos almacenados y generación de un informe final de migración.
